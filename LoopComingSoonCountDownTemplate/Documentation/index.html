HTML Structure

This Template is a fixed layout. All of the information within the main content area is nested within a div with an id of "container main-wrapper". The general template structure is the same throughout the template. Here is the general structure.

If you would like to change color of circle countdown OR Need to change Title of Days, Hours etc.. Please Open main.js file and find below code and edit it

bgColor: "rgba(22, 34, 0, 0.8)",
fgColor: "#a4ff00",
fgLast: "#bafc00",

JavaScript

This Template imports Below Javascript files.

  1. Jquery - Main Javascript Library
  2. Modernizr - JavaScript library that detects HTML5 and CSS3 features in the user’s browser.
  3. ClassyCountdown - Countdown Script
  4. Backstretch - Full Size Background Image and Slideshow
  5. Tubular - Full Size Background Video
  6. Knob
  1. To change the launch date, open iny index page and change the date in the data-end="yourvalue". e.g. 
    //enter the count down date using the format month day, year hours:minutes:seconds
"<div class="countdown" data-end="Your date and time"></div>"
